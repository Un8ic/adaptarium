# АДАПТАРИУМ

Цифровой сервис для адаптации персонала "АДАПТАРИУМ" - это инновационное решение для сокращения текучести кадров и ускорения выхода новых сотрудников на плановые показатели.

## Структура проекта

- `index.html` - главная страница
- `css/` - стили
  - `style.css` - основные стили
  - `responsive.css` - адаптивные стили
- `js/` - JavaScript файлы
  - `utils.js` - утилитарные функции
  - `auth.js` - аутентификация
  - `navigation.js` - навигация
  - `materials.js` - материалы для обучения
  - `games.js` - игры и обучение
  - `tests.js` - тесты и комментарии
  - `profile.js` - профиль пользователя
  - `app.js` - основной файл приложения

## Функциональность

- Аутентификация пользователей (администраторы и менеджеры)
- Материалы для обучения
- Интерактивные игры и тесты
- Система комментариев и отзывов
- Чат для общения
- Аналитика для администраторов
- Адаптивный дизайн
